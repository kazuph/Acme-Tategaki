# NAME

Acme::Tategaki - It makes a text vertically.

# SYNOPSIS

    $ tategaki お前は、　すでに、　死んでいる。
    死　す　お
    ん　で　前
    で　に　は
    い　、　、
    る
    。

# DESCRIPTION

Acme::Tategaki makes a text vertically.
